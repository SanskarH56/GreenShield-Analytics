Conservation Risk Report

This page explains how the dashboard converts forest-cover and biodiversity data into practical conservation intelligence. Use it during presentation to explain the problem, methodology, limitations, and recommended actions.

← Back to Dashboard Open Spatial Map

1. Project Purpose

Deforestation affects biodiversity, habitat stability, and long-term ecosystem health. GreenShield Analytics helps identify which ecological zones need the most urgent attention by combining multiple environmental indicators into one decision-support view.

2. Data Used

  • Region-wise forest area values
  • Year-wise biodiversity index
  • Species-count indicators where available
  • Derived year-over-year percentage change

3. Priority Score Methodology

The dashboard uses a composite score so regions can be ranked by intervention urgency instead of looking at separate values one by one.

Priority Score =
0.5 × |Forest Change %| +
0.3 × |Biodiversity Change %| +
0.2 × (100 − Biodiversity Index)

4. Severity Interpretation

  • Stable: Maintain protection and continue monitoring.
  • Warning: Watch closely and prepare preventive action.
  • Critical: Prioritize immediate conservation intervention.

5. Dashboard Features

  • Interactive India ecological risk map
  • Trend chart with Top 5 risk filtering
  • Priority ranking and alert system
  • Region comparison and detail panel
  • Downloadable conservation summary

6. Limitations

The project uses historical CSV data and representative map coordinates. It demonstrates environmental analytics concepts, but it is not a real-time government monitoring system or legally precise GIS boundary map.

7. Future Scope
